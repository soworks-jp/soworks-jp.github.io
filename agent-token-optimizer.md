# Agent Token Optimizer

Agent Token Optimizer helps inspect repository context before sending work to AI agents.

It is for reducing context waste, spotting large or risky files, and giving model routing hints before expensive or fragile agent runs.

## Who it is for

- Developers working in large repositories
- AI agent operators who need context budgeting
- Maintainers who want preflight reports before task handoff

## What it provides

- Repository scanning
- File classification
- Token budget signals
- Model routing hints
- Reports for agent handoff
