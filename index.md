# SO Works OSS Guide

Local-first OSS tools for AI agents, reusable skills, review workflows, and developer productivity.

These projects are designed for people who use Codex, Claude Code, Gemini CLI, GitHub Copilot, Cursor, Continue, and other AI coding assistants in real repositories.

## Projects

| Project | What it does | Status |
|---|---|---|
| [Agent Workbench](agent-workbench.md) | Local task board, routing, claims, prompts, reports, and PR briefs for AI coding agents | pre-public |
| [Agent Skill Shelf](agent-skill-shelf.md) | Manage reusable AI-agent skills as source bundles and generate platform-specific outputs | pre-public |
| [Agent Token Optimizer](agent-token-optimizer.md) | Inspect repository context, token risk, and model routing signals before agent work | pre-public |
| [Fusion Skill](fusion-skill.md) | Add multi-perspective review and judge synthesis to AI agents | pre-public |

## Languages

- English pages are short product summaries and links to the canonical README.
- [Japanese pages](ja/index.md) explain usage patterns, Japanese prompts, and operational cautions in more detail.
