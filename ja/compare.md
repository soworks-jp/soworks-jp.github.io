# 比較

| やりたいこと | 使うOSS |
|---|---|
| AI coding agentのタスク、claim、prompt、report、PR briefを管理したい | Agent Workbench |
| Codex / Claude Code / Gemini CLI / Copilot / Continue / Antigravity向けのskillをまとめて管理したい | Agent Skill Shelf |
| agentに渡す前にrepoのcontext量、巨大ファイル、tokenリスクを確認したい | Agent Token Optimizer |
| ひとつの回答を複数視点でレビューし、judgeで統合してから出したい | Fusion Skill |

## 使い分け

- 作業をAI agentへ渡す前のタスク管理なら Agent Workbench。
- 再利用するpromptやskillを管理するなら Agent Skill Shelf。
- 大きいrepoで読む範囲を絞るなら Agent Token Optimizer。
- 仕様、PRD、文章、レビュー結果を強くするなら Fusion Skill。

各OSSは小さく、ローカルファーストに保つ。ホスティング、独自ドメイン、有料インフラは、OSSポートフォリオの反応が出てから追加する。
