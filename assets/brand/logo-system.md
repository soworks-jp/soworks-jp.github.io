# SO Works Logo System

SO Works uses a compact S-shaped workflow mark for OSS guide pages, profile icons, favicons, and social previews.

## Why This Logo

The mark is intentionally not a literal robot, chat bubble, code bracket, or Japanese flag. SO Works is meant to be a durable OSS and work-system brand, not a single AI-chat product. The logo therefore uses a simple abstract workflow symbol that can keep working across AI tools, local developer utilities, docs sites, and future OSS packages.

The current mark combines three ideas:

- `S` path: stands for SO Works and keeps the icon readable even without the wordmark.
- Workflow route: the curved stroke suggests task flow, handoff, review, and iteration.
- Three nodes: represent agent, human, and repository context working together.

The color roles are also intentional:

- Ink `#172033`: serious developer-tool base, usable on GitHub Pages, docs, and app icons.
- Paper `#F8FAFC`: high-contrast stroke that stays legible at favicon size.
- Signal blue `#38BDF8`: AI/system signal and routing.
- Work orange `#F97316`: active work, execution, and handoff.
- Growth green `#22C55E`: completion, validation, and release readiness.

## Alternatives Considered

- `SW` monogram: clear as initials, but too generic and weaker at favicon size.
- Gear or wrench: too close to generic tooling and maintenance apps.
- Robot face: too trend-bound and too narrow for an OSS organization brand.
- Chat bubble: implies conversation products, while SO Works is broader than chat.
- Globe or flag motif: would overemphasize region instead of OSS and workflow.

## Design Constraints

- Must work at 16px to 32px for browser tabs and GitHub UI.
- Must be original and avoid copying existing AI assistant, cloud, or developer-tool marks.
- Must remain recognizable in a single square icon without the `SO Works` wordmark.
- Must feel quiet and technical enough for OSS docs, not like a consumer social app.
- Must support later variants for product icons without changing the core mark.

## Assets

- `icon.svg`: primary square icon for public profiles and app surfaces.
- `favicon.svg`: small-size browser favicon.
- `assets/brand/soworks-mark.svg`: reusable source mark.
- `assets/brand/soworks-lockup.svg`: mark plus SO Works wordmark.

## Colors

- Ink: `#172033`
- Paper: `#F8FAFC`
- Signal blue: `#38BDF8`
- Work orange: `#F97316`
- Growth green: `#22C55E`

## Usage

- Use the square mark when the available size is below 96px.
- Use the lockup when there is enough horizontal space.
- Keep clear space of at least 12.5% of the icon width around the mark.
- Do not place the mark on dark patterned backgrounds without a solid container.
- Do not recolor individual nodes unless a new brand variant is defined.
