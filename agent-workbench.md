# Agent Workbench

Agent Workbench is a local workbench for AI coding agents.

It turns repository TODOs, issue packs, roadmap items, missing tests, and review needs into local tasks that can be routed to Codex, Claude Code, Gemini CLI, or other agents.

## Who it is for

- Developers using multiple AI coding agents
- OSS maintainers who want safer task handoff
- Teams that want local task evidence before PRs

## What it provides

- Task scouting
- Agent routing
- Claim / lease files to reduce collisions
- Agent prompts
- Work reports
- PR briefs
- Review queue entries
