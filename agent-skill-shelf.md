# Agent Skill Shelf

Agent Skill Shelf is a local-first toolkit for managing reusable AI-agent skills.

It treats `skills/<name>/` as the source of truth and generates platform-specific outputs for Codex, Claude Code, Gemini CLI, GitHub Copilot, Continue, and Antigravity.

## Who it is for

- AI power users with many reusable prompts
- Teams that want skill review before installation
- Maintainers who need platform-specific skill outputs without copy drift

## What it provides

- Source bundle authoring
- Project-local sync
- Inventory
- Static security scan
- Discovery and improvement reports
- Markdown / JSON / HTML viewers

The examples use fixture skill names, generic paths, and rounded status data. Real skill reports should be published only after internal paths, secret-risk concrete values, private skill names, hostnames, and private repository identifiers are removed.
