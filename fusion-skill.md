# Fusion Skill

Fusion Skill adds multi-perspective review and judge synthesis to AI agents.

It is a local, prompt-based skill. It does not require an API key or a SaaS account. It is inspired by general Fusion-style reasoning patterns, but it is not affiliated with OpenRouter and does not implement OpenRouter Fusion Router.

## Who it is for

- Developers who want stronger final answers from AI agents
- PMs and founders reviewing specs, ideas, and plans
- OSS maintainers who want reusable review roles

## What it provides

- Multi-role review
- Judge synthesis
- Risk and skeptic review
- Codex-ready implementation instruction generation
- Agent-specific install targets
